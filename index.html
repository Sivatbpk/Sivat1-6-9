ประเภทของเซิร์ฟเวอร์

เรียนรู้เกี่ยวกับเซิร์ฟเวอร์ที่สำคัญในระบบเครือข่าย

Database Server

เครื่องแม่ข่ายฐานข้อมูล

หน้าที่หลัก

ให้บริการจัดเก็บ จัดการ และเข้าถึงข้อมูลในฐานข้อมูลแก่ผู้ใช้งานหรือแอปพลิเคชัน

ซอฟต์แวร์ที่ใช้บ่อย

  • MySQL / MariaDB
  • PostgreSQL
  • Oracle Database
  • Microsoft SQL Server

ตัวอย่างการใช้งาน

ระบบจัดเก็บข้อมูลลูกค้าในเว็บแอปพลิเคชัน, ข้อมูลสินค้า, ข้อมูลนักเรียนในโรงเรียน

Mail Server

เครื่องแม่ข่ายจดหมายอิเล็กทรอนิกส์

หน้าที่หลัก

รับ ส่ง และจัดเก็บจดหมายอิเล็กทรอนิกส์ (Email) ระหว่างผู้ใช้งานภายในและภายนอกเครือข่าย

ซอฟต์แวร์ที่ใช้บ่อย

  • Postfix (Mail Transfer Agent - MTA)
  • Microsoft Exchange Server
  • Sendmail
  • Zimbra Mail Server

ตัวอย่างการใช้งาน

ระบบอีเมลภายในองค์กร เช่น ชื่อผู้ใช้@ชื่อบริษัท.com, ระบบแจ้งเตือนผ่านอีเมลจากระบบต่าง ๆ

DNS Server

เครื่องแม่ข่าย DNS

หน้าที่หลัก

แปลงชื่อโดเมน (เช่น www.example.com) ให้เป็นหมายเลข IP (เช่น 192.0.2.1) ที่เครื่องคอมพิวเตอร์สามารถเข้าใจได้

ซอฟต์แวร์ที่ใช้บ่อย

  • BIND (Berkeley Internet Name Domain)
  • Microsoft DNS Server
  • Unbound
  • PowerDNS

ตัวอย่างการใช้งาน

ใช้ในระบบเครือข่ายเพื่อให้ผู้ใช้สามารถเข้าถึงเว็บไซต์และบริการต่าง ๆ โดยใช้ชื่อโดเมนแทน IP
